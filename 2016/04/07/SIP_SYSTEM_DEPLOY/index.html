2 rtpproxy的NAT穿透

穿透的技术很多，stun ice 我都没搞懂呢。kamilio用的是rtpproxy，先试用下这个了。
官方支持的NAT穿透就是用这个，必须自己编译和安装。

2.1 编译

  • 官方当前最新 rtpproxy-2.0.0 ，从github上下载，官网貌似没有看到什么内容啊。

  • 直接configure make make install 。

  • 源码安装得到没有配置文件，不用找了。程序在：
    libtool: install: /usr/bin/install -c rtpproxy_debug /usr/local/bin/rtpproxy_debug

2.2 运行rtpproxy

服务器配置文件要求这么使用：

- install RTPProxy: http://www.rtpproxy.org
 - start RTPProxy:
  rtpproxy -l _your_public_ip_ -s udp:localhost:7722
- option for NAT SIP OPTIONS keepalives: WITH_NATSIPPING
